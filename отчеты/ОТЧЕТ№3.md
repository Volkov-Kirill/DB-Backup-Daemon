## 2. Инструкция по быстрому запуску

### Требования

Для запуска проекта необходимо установить:

* Windows 10/11
* Visual Studio 2022
* .NET Framework 4.7.2 Developer Pack

---

### Клонирование репозитория

```bash
git clone https://github.com/Volkov-Kirill/DB-Backup-Daemon.git
cd DB-Backup-Daemon
```

---

### Настройка `.env`

В корне проекта необходимо создать файл `.env`.

Пример содержимого:

```env
BACKUP_DB_PATH=database.db
BACKUP_DIRECTORY=backups
BACKUP_PASSWORD=12345
BACKUP_INTERVAL_MINUTES=1
APP_DATA_PATH=appdata.db
```

### Описание переменных

| Переменная                | Назначение                              |
| ------------------------- | --------------------------------------- |
| `BACKUP_DB_PATH`          | Путь к файлу базы данных                |
| `BACKUP_DIRECTORY`        | Папка для хранения ZIP-архивов          |
| `BACKUP_PASSWORD`         | Пароль для защищенного архива           |
| `BACKUP_INTERVAL_MINUTES` | Интервал автоматического запуска бэкапа |
| `APP_DATA_PATH`           | Локальное хранилище истории операций    |

---

### Запуск через Visual Studio

1. Открыть файл решения:

```text
DB-Backup-Daemon.sln
```

2. Собрать проект:

```text
Ctrl + Shift + B
```

3. Запустить проект:

```text
F5
```
